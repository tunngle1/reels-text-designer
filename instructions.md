
# Инструкция для разработчика: Reels Text Designer

## Как добавить новые шрифты в проект (статически)

Если вы хотите добавить новые Google Fonts в предустановленный список:
1. Откройте `index.html`.
2. Найдите секцию `<link href="https://fonts.googleapis.com/css2?...">`.
3. Добавьте нужное семейство шрифтов из Google Fonts. Обязательно выбирайте те, которые поддерживают **Cyrillic**.
4. Откройте `constants.ts`.
5. Добавьте новый объект в массив `INITIAL_FONTS`:
   ```typescript
   { 
     id: 'unique_id', 
     name: 'Название для списка', 
     family: 'Название шрифта из CSS (family)', 
     source: 'google' 
   }
   ```

## Как добавить локальные .ttf файлы
Чтобы включить файлы шрифтов напрямую в код (без загрузки пользователем):
1. Конвертируйте ваш `.ttf` файл в формат **Base64** (можно использовать любой онлайн-конвертер).
2. Вставьте эту строку в `INITIAL_FONTS` в `constants.ts`, указав `source: 'custom'` и `url: 'data:font/ttf;base64,...'`.
3. Приложение автоматически зарегистрирует его через `FontFace` API.

## Запуск проекта
Приложение полностью клиентское. Для запуска достаточно:
1. `npm install`
2. `npm start`

Для развертывания в Telegram:
- Создайте бота в @BotFather.
- Настройте Web App, указав URL вашего хостинга (например, Vercel).
- Приложение будет использовать CSS-переменные Telegram для автоматической адаптации под тему (темную или светлую).
